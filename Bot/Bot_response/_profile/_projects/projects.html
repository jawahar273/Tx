Open Source Project

  • PNLT

    Practical Natural Language Processing Tools for Humans is build on the top of Senna Natural Language Processing (NLP) predictions: part-of-speech (POS) tags, chunking (CHK), name entity recognition (NER), semantic role labeling (SRL) and syntactic parsing (PSG) with skip-gram all in Python and still more features will be added. The website give is for downloading Senna tool

  • Tx project(work in progress)

    Tx is an umbrella package with compressed of Tx(based on Sanic) and TxBot NLU Bot(Snip-nlu). The primary aim of this project is intergate the bot in our daily task.

    A budget manager which is build in django and DRF
